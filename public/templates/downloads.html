Desarrollo en React Native

Quiago ha sido desarrollado utilizando React Native, una poderosa plataforma que nos permite crear aplicaciones móviles multiplataforma con un rendimiento nativo. Nuestra aplicación combina la flexibilidad de React con la potencia de las plataformas nativas.

Disponible para iOS

Actualmente, Quiago está disponible para dispositivos iOS. Hemos trabajado arduamente para optimizar la experiencia en Swift, asegurando un rendimiento fluido y una interfaz intuitiva para los usuarios de iPhone. Disfruta de una navegación suave y descubre nuevos lugares en Bolivia con facilidad.

Próximamente en Android

¡Buenas noticias para los usuarios de Android! Estamos trabajando diligentemente para traer Quiago a tu dispositivo favorito. Pronto podrás explorar Bolivia y planificar tus aventuras con Quiago en cualquier smartphone Android. ¡Mantente atento a nuestras actualizaciones!
